01 - Daily Three 🌤️
A simple daily goal tracker that helps you focus on three tasks a day.
💡 What it does
Set 3 goals for the day
Check off goals as you complete them
Progress bar fills as you complete each goal
Goals are saved and persist after page refresh
🛠️ Built With
HTML
CSS
JavaScript (Vanilla)
localStorage
📚 Concepts Practiced
DOM manipulation
Event listeners (change, input)
localStorage — saving and loading data
JSON.stringify and JSON.parse
CSS positioning (absolute, relative)
Custom checkbox styling with appearance: none
Progress bar with dynamic width using JS
🚀 How to Run
Open index.html in your browser. No installations needed.
